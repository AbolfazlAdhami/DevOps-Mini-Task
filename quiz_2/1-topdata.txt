top => command up =>  مدت زمان  روشن بودن سیستم , users => چه تعداد کاربر لاگین هستند
load average => میانگین بار پردازنده در 1 5 15 دقیقه گذشته
0.35 => in last 1 min
0.94 => in last 5 min
1.26 => in last 15 min
task => تعداد پردازش کلی total, running در حال اجرا ,stopped متوقف شده,zombie زامبی پردازش های که خودش تمام شده ولی پدرش هنوز نه,
%Cpu(s): us : User درصدی که برای کار های کاربر پردازش شده,sy: System درصدی که در کرنل مصرف شده ,
	 ni:nice درصد زمانی برای پردازش هایی که priority تغییر کرده , id idle  درصد بیکاری پردازنده
	 wa: I/O wait منتظر عملیات دیسک شبکه
	 hi: hardware interputs
	 si: software interputs
	 st: Steal زمانی ماشین مجازی که پردازنده در سیستم میزبان گرفته 
Mib Mem : وضعیت رم ,total مقدار کلی, free مقدار ازاد , used استفاده ,buff/cache کش و بافر
Mib Swap : مقدار حافظه دیسک برای تامین نیاز رم 

PID شماره تسک پردازش شده 
USER کاربر که پردازش را اجرا کرده
PR اولویت تسک ها عدد کمتر اولویت بیشتر priority
NI Nice Value -> niceness (min:-20 max:19)
VIRT Vertual Mem حافطه مجازی برا  پردازش رزرو 	
RES حافظه فیزیکی که پردازش واقعا استفاده کرده
SHR share حافظه شده با تسک های دیگر
S وضعیت پردازنده R Running S Sleeping T Stopped Z Zombie
CPU%  درصد استفاده در بازه رفرش
MEM% درصد کل استفاده از رم
+Time کل زمانی که پردازنده پردازش کرده
Command نام و دستور اجرا شده
