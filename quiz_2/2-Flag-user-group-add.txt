// Flag useradd , groupadd  //

useradd [options] USERNAME 
-c  "Comment" توضیحات برای کاربر ساخته شده
-m  ساخت دایرکتوری اگر وجود نداشته باشد
-d /path/to/home  دایرکتوری اختصاصی 
-s /bin/bash  set login shell
-M دایرکتوری در home ساخته نشود
-G group1,group2  عضویت در یک گروه 
-u UID اختصاص یه ای دی به یوزر
-e YYYY-MM-DD تاریخ انقضا برای کاربر
-f days تعداد روز ها که پس از انقضا پسسورد تا غیرفعال شدن حساب
-r ساخت حساب سیستمی بدون لاگین شل و تاریخ انقضا
-p PASSWORD تنظیم یه پسوورد رمز نگاری شده 

groupadd [OPTIONS] GROUPNAME

-g GID ست کردن یه ایدی اختصاصی
-r ساخت یه گروه سیستمی
-f اگر گروه وجود داشته باشد از ان خارج و تغییری ایجاد نمی کند
-K KEY=VALUE اورراید فایل /etc/login.defs

Examples:

useradd -m -s /bin/bash -g developer -G sudo user1
passwd user1

useradd -r -M -s /usr/sbin/nologin backupuser

groupadd -g 1500 devops
